设置环境变量

[bash]修改单用户环境变量文件

sudo vi ~/.bashrc

[bash]修改所有用户环境变量文件

sudo vi /etc/profile

root权限和用户权限切换

切换至root权限

sudo su

切换至用户权限

su <username>

修改当前账号(root或普通用户)的密码

passwd

修改指定用户密码

passwd <username>

服务器文件传输

本地向服务器文件传输

scp -r <uesrname>@<服务器ip>：<服务器指定文件夹路径> <本地所存文件路径>

服务器文件下载到本地

scp -r  <本地所存文件路径>  <uesrname>@<服务器ip>：<服务器指定文件夹路径>

ssh服务

安装ssh

sudo apt-get install openssh-server

启动ssh服务

service sshd start

查看ssh服务是否开启

ps -e | grep ssh

通过ssh远程访问服务器

ssh <username>@<ip>

查看系统信息

查看系统版本信息

uname -a

修改主机名

vi /etc/hostname

查看英伟达GPU型号

lspci | grep -i nvidia

查看NVIDIA驱动版本

sudo dpkg --list | grep nvidia-*

sudo apt install python3-pip
